# CRM & Measurement Dashboard for Tailor Shop

## 프로젝트 개요

본 프로젝트는 50년 전통 양복점에서 실제 사용 중이던  
종이 문서와 엑셀 기반의 회원·치수·주문 데이터를 정리하고,  
이를 웹 기반 대시보드로 통합 관리하기 위해 개발된 시스템입니다.

별도의 프로그램 설치 없이 웹 브라우저에서 바로 사용할 수 있으며,  
기존 업무 방식을 최대한 유지하면서도  
데이터를 체계적으로 축적·활용할 수 있도록 설계되었습니다.

---

## 해결하고자 한 문제

- 회원 정보가 종이, 엑셀, 기억에 분산되어 관리됨
- 고객의 과거 치수 이력을 한눈에 파악하기 어려움
- 최근 치수가 아닌 예전 치수로 재단되는 실수 가능성
- 원단 사용량과 출고 기록을 수작업으로 관리
- 숙련자의 판단 기준이 개인 경험에 의존

---

## 주요 기능

### 1. 회원 관리 (CRM)

- 회원 고유 ID 자동 생성
- 이름 또는 회원번호로 빠른 검색
- 기존 엑셀 회원정보 자동 정규화 및 병합
- 전화번호 형식 자동 정리 및 메모 분리

### 2. 치수 이력 관리

- 회원별 치수 측정 기록을 날짜 기준으로 저장
- 여러 건의 치수 중 가장 최근 데이터 자동 선택
- 과거 치수 변화 이력 조회 가능

### 3. 치수 시각화

- 자켓 및 바지 이미지 위에 주요 치수 직접 표시
- 숫자만 보는 방식이 아닌 직관적인 상담 지원
- 최근 치수 기준으로 자동 시각화

### 4. 자동 사이즈 참고 제안

- 입력된 치수를 기반으로 일반적인 사이즈 범위 표시
- 재단 판단을 보조하는 참고용 기능

### 5. 원단 관리 자동화

- 주문 데이터 기반 원단 사용량 계산
- 자동 원단 출고 처리
- 재고 이동 내역 기록

---

## 데이터 활용 방식

- 회원 데이터: 고객 관리 및 이력 추적
- 치수 데이터: 재단 참고 및 체형 변화 분석
- 주문 데이터: 원단 사용량 계산 및 재고 관리
- 누적 데이터: 향후 체형 유사 고객 분석 및 상담 보조

---

## 기대 효과

- 종이·기억 중심 업무에서 데이터 기반 업무로 전환
- 재단 및 상담 과정에서의 실수 감소
- 직원 간 노하우 공유 기반 마련
- 고객 응대의 일관성과 신뢰도 향상

---

## 향후 확장 방향

- 유사 체형 고객 추천
- 치수 변화 패턴 분석
- 재방문 주기 분석
- 재단 상담 보조 모델로 확장 가능

---
